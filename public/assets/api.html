      /rest-auth/login/

      Route for users login to the system. Environment variable: REACT_APP_ENDPOINT_AUTH_LOGIN

      /rest-auth/login/ post

      post

      Retrieve user's login token key

      post /rest-auth/login/

      Retrieve user's login token key

      • Request
      • Response

      Body

      Media type: application/json

      Type: object

      Properties
      • email: required(string)
      • password: required(string)

      Example:

      {
    "email": "testuser0@shop.aa",
    "password": "pass0"
}

      HTTP status code 200

      OK

      Body

      Media type: application/json

      Type: any

      Example:

      {
  "key": "zx0e2eeb00d740d4582682d4b690f0823dc9ac28"
}

      HTTP status code 400

      Bad Request

      /rest-auth/registration/

      Manages user's registration. Environment variable: REACT_APP_ENDPOINT_SIGNUP

      /rest-auth/registration/ post

      post

      Retrieve user's login token key

      post /rest-auth/registration/

      Retrieve user's login token key

      • Request
      • Response

      Body

      Media type: application/json

      Type: object

      Properties
      • username: required(string)

        Must be unique.

      • email: required(string)

        Must be unique.

      • password1: required(string)
      • password2: required(string)

        Repeat password.

      Example:

      {
    "username": "testuser0",
    "email": "testuser0@shop.aa",
    "password1": "pass0"
    "password2": "pass0"
}

      HTTP status code 201

      Created. New user created

      HTTP status code 400

      Bad Request. Error due to missing required field or adding username/email already on the DB. Username and email must be unique.
